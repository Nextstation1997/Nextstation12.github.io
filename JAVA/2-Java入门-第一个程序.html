# 在IntelliJ IDEA中创建项目

打开 IDEA -> Create New Project

选择 java -> Project SDK(1.8版本) -> Next

选择项目名称和工作空间，然后点击 Finish 即可创建成功

右击 src(源码目录) -> New -> Java Class -> 输入文件名称 demo

输入以下代码

    public static void main(String[] args) {
        System.out.println("Hello World!");
    }

点击上方工具栏 Run(绿色箭头) 运行程序,此时 我们的第一个程序Hello World！就写好了

# 编写java程序注意事项

  • Java对大小写敏感，如果出现了大小写拼写错误，程序无法运行。

  • 关键字public被称作访问修饰符（access modifier）,用于控制程序的其它部分对这段代码的访问级别。

  • 关键字class 表明Java 程序中的全部内容都包含在类中，Java是一种纯面向对象的语言。

  • 一个源文件中至多只能有一个public的类声明，其它类的个数不限，如果源文件中包含一个public 类，源文件名必须和它中定义的public的类名相同，且以java为扩展名。

  • 一个源文件可以有多个class

  • 正确编译后的源文件，会得到相应的字节码文件，编译器为每个类生成独立的字节码文件，且将字节码文件自动命名为类的名字且以class为扩展名。

  • 在Java中，用花括号划分程序的各个部分，任何方法的代码都必须以{开始，以}结束， 由于编译器忽略空格，所以大括号风格不受限制。

  • Java中每个语句必须以分号结束，回车不是语句的结束标志，所以一个语句可以跨多行。
